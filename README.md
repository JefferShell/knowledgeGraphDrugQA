## 中草药知识问答系统

### 项目简介

中草药知识问答系统是一个基于知识图谱的智能平台，专注于提供全面、准确的中草药相关知识。通过深度整合大量优质数据资源，并利用先进的图谱展示与查询技术，为用户呈现一个直观、高效的知识探索环境。此外，系统还集成了自然语言处理功能，支持用户以自然语言形式进行提问，进一步提升了用户体验。


### 主要功能

* **知识图谱展示与查询：**直观的知识图谱展示，可视化展示中草药之间的关系。支持节点关系查询，快速获取与之相关联的其他节点信息。
* **知识问答：**强大的自然语言识别能力，可以准确解析用户提出的问题，并生成简洁明了的回答。


### 技术架构

- **前端：**echarts
- **后端：**py2neo + Neo4j数据库
- **自然语言处理：**jieba分词技术
